# Astro Forecast

A beautiful astronomy and astrology forecast web app powered by Google's Gemini AI. Get personalized horoscopes, track moon phases, and discover significant astrological events.

## Features

- **Daily Horoscope**: Personalized readings based on your zodiac sign
- **Moon Phase Tracker**: Real-time moon phase calculations with visual representations
- **Astro Events**: Daily significant astrological events and their meanings
- **Multiple Zodiac Signs**: Track your Sun, Moon, and Rising signs
- **AI-Powered Insights**: All content generated by Gemini AI for unique, personalized readings
- **Beautiful Dark Theme**: Cosmic-inspired UI with animated starfield background
- **Privacy-First**: Your API key is stored locally in your browser

## Setup Instructions

### 1. Get a Gemini API Key

1. Visit [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Sign in with your Google account
3. Click "Create API Key"
4. Copy your API key

### 2. Deploy to GitHub Pages

#### Option A: Using GitHub Web Interface

1. Create a new **private** repository on GitHub named `astro-forecast`
2. Upload all files from this project:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `README.md`
3. Go to repository Settings > Pages
4. Under "Source", select "Deploy from a branch"
5. Select the `main` branch and `/ (root)` folder
6. Click "Save"
7. Your site will be published at `https://yourusername.github.io/astro-forecast/`

#### Option B: Using Git Command Line

```bash
# Navigate to the project directory
cd astro-forecast

# Add all files
git add .

# Create initial commit
git commit -m "Initial commit: Astro Forecast app"

# Create a new private repository on GitHub, then:
git remote add origin https://github.com/yourusername/astro-forecast.git

# Push to GitHub
git push -u origin main
```

Then follow steps 3-7 from Option A to enable GitHub Pages.

### 3. Configure the App

1. Open your deployed app
2. Click the settings icon (⚙️) in the top right
3. Paste your Gemini API key
4. Click "Save Settings"
5. Select your zodiac signs (Sun, Moon, Rising)

Your API key is stored securely in your browser's local storage and is never sent anywhere except directly to Google's Gemini API.

## Technology Stack

- **Frontend**: Vanilla JavaScript, HTML5, CSS3
- **AI**: Google Gemini Pro API
- **Hosting**: GitHub Pages
- **Storage**: Browser LocalStorage for settings

## Privacy & Security

- Your API key is stored only in your browser's local storage
- No data is collected or sent to any third-party servers
- All AI requests go directly from your browser to Google's Gemini API
- The app works entirely client-side

The app is built with a modular structure:
- `loadHoroscope()`: Handles horoscope generation
- `loadAstroEvents()`: Handles astro events
- `loadMoonInfo()`: Handles moon information
- `calculateMoonPhase()`: Calculates current moon phase

## Troubleshooting

### API Key Issues
- Make sure your API key is correct
- Check that you've enabled the Gemini API in your Google Cloud Console
- Verify you haven't exceeded your API quota

### Content Not Loading
- Check your browser's console for errors (F12)
- Ensure you have a stable internet connection
- Try clearing your browser cache and re-entering your API key

### GitHub Pages Not Working
- Make sure GitHub Pages is enabled in repository settings
- Check that you've selected the correct branch (usually `main`)
- It may take a few minutes for changes to go live
